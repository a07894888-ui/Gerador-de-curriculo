Projeto da Atividade Prática Orientada (APO) – Fundamentos de Programação para Internet

 Descrição do Projeto

Este projeto foi desenvolvido como parte da Atividade Prática Orientada (APO) da disciplina Fundamentos de Programação para Internet, com o objetivo de criar uma aplicação web funcional que gere currículos automaticamente a partir das informações inseridas pelo usuário.

O sistema foi construído utilizando PHP, HTML, CSS, JavaScript e Bootstrap (via CDN do jsDelivr).

 Funcionalidades
	•	Formulário para inserção de dados pessoais;
	•	Cálculo automático da idade;
	•	Adição dinâmica de experiências profissionais;
	•	Geração do currículo em nova aba;
	•	Botão para imprimir ou baixar o currículo.

 Tecnologias Utilizadas
	•	HTML5
	•	CSS3
	•	JavaScript
	•	jQuery
	•	Bootstrap 5 (via jsDelivr)
	•	PHP
  Como Executar o Projeto

  1.	Extraia a pasta curriculum-generator para:
     C:\xampp\htdocs\

  2.Inicie o Apache no XAMPP.
  
	3.No navegador, acesse:
    http://localhost/curriculum-generator/index.php4.	Preencha o formulário e clique em “Gerar Currículo”.
